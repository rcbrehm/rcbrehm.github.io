# Robin Christopher Brehm

**Predoctoral University Assistant**, Department of East Asian Studies, University of Vienna. Vice-Director, European Centre for North Korean Studies (ECNK).

This repository is the basis of my personal website at [rcbrehm.github.io](https://rcbrehm.github.io).

Built with Jekyll, hosted on GitHub Pages.

## Updating

- Personal details and sidebar links: `_config.yml`
- Page texts: `index.md` (About), `projects.md`, `research.md`, `teaching.md`, `cv.md`
- Headshot: replace `assets/images/headshot.jpg`
- CV: place the PDF at `assets/pdf/brehm-cv.pdf` and set `cv_pdf: true` in `_config.yml`
